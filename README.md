🪔 Diwali Sales Analysis
📌 Project Overview

This project analyzes Diwali sales data to understand customer behavior, purchasing patterns, and business insights. The goal is to help businesses make better data-driven decisions during festive seasons.

📂 Dataset

The dataset includes the following features:

User ID
Gender
Age Group
State
Occupation
Product Category
Amount (Purchase)
🛠️ Technologies Used
Python 🐍
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook
🔍 Project Workflow
1. Data Cleaning
Removed null values
Dropped unnecessary columns
Converted data types
2. Exploratory Data Analysis (EDA)
Gender-wise analysis
Age group analysis
State-wise sales
Occupation insights
Product category performance
3. Data Visualization
Bar charts
Count plots
Distribution plots
📊 Key Insights
👩 Females have higher purchasing power than males
👨‍👩‍👧 Age group 26–35 years contributes most to sales
🏙️ Top purchasing states include Uttar Pradesh, Maharashtra, Karnataka
💼 Working professionals & IT sector customers spend more
🛍️ Most sold categories: Clothing, Electronics, Food
📈 Conclusion
Targeting young adults (26–35) can increase sales
Focus marketing on female customers
Apply region-based strategies to maximize revenue
